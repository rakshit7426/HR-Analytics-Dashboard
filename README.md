# HR Analytics Dashboard

## Project Overview

An interactive Power BI dashboard built to analyze workforce, compensation, employee satisfaction, and attrition trends, helping HR teams make data-driven decisions.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization

## Dashboard Pages

### 1. Workforce Overview
- Total Employees
- Average Salary
- Average Experience
- Average Age
- Gender Distribution
- Department-wise Employee Analysis
- Salary Slab Analysis
- Educational Field Analysis
- Job Role Analysis

### 2. Compensation Overview
- Average Salary
- Salary Distribution
- Job Role vs Average Salary
- Salary vs Experience
- Job Level Distribution

### 3. Employee Satisfaction Analysis
- Job Satisfaction Metrics
- Work-Life Balance Analysis
- Employee Engagement Insights

### 4. Attrition Analysis
- Attrition Rate
- Attrition by Department
- Attrition by Job Role
- Attrition by Age Group
- Attrition Trends

## Features

- Multi-Page Dashboard
- Navigation Buttons
- KPI Cards
- Interactive Slicers
- Dynamic Visualizations

## Skills Demonstrated

- Data Cleaning using Power Query
- Data Modeling
- DAX Calculations
- Dashboard Design
- Data Analysis & Reporting
- Data Visualization
