# HR Analytics Dashboard ## Project Overview This Power BI dashboard provides insights into workforce composition, employee satisfaction, compensation trends, and attrition analysis. The dashboard helps HR teams make data-driven decisions by analyzing employee-related metrics. ## Tools Used - Power BI - Power Query - DAX - Data Visualization ## Dashboard Pages ### 1. Workforce Overview - Total Employees - Average Salary - Average Experience - Average Age - Gender Distribution - Department-wise Employee Analysis - Salary Slab Analysis - Educational Field Analysis - Job Role Analysis ### 2. Compensation Analysis - Salary Distribution - Compensation Insights - Department-wise Salary Analysis ### 3. Employee Satisfaction Analysis - Job Satisfaction Metrics - Work-Life Balance Analysis - Employee Engagement Insights ### 4. Attrition Analysis - Attrition Rate - Attrition by Department - Attrition by Job Role - Attrition Trends ## Features - Multi-Page Dashboard - Navigation Buttons - KPI Cards - Dynamic Visualizations ## Skills Demonstrated - Data Cleaning using Power Query - DAX Calculations - Dashboard Design - Data Analysis & reporting - Data Visualization ## Dashboard Screenshots Dashboard Screenshots are included in this repository. ## Author Rakshit Kumar
